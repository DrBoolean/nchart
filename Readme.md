nchart
==========

nChart for node.js inspired by [Chart.js][].

## Installation

  $ npm install -g nchart

## Features

  * Built on [node-canvas][]

## Charts

  * Polar area chart
  * Doughnut chart
  * Pie chart
  * Radar chart
  * Bar chart
  * Line chart
  

[Chart.js]: http://www.chartjs.org/
[node-canvas]: https://github.com/LearnBoost/node-canvas
